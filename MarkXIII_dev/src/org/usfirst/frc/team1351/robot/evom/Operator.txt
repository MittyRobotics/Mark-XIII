Driver Controls:
- Right/Left Bumpers on xbox: Intake/Outtake Respectively  
- Xbox, Select: Switch Camera  
- Triggers on xbox: Variable% speed for each respective side 
- Xbox, Button B: breach portcullis
- Xbox, Button Y: breach cheval
- Xbox, Strong Vibration on Full: Alerts to current overdraw by any talons  

For Testing:

- Stick 1, Button 3: toggle intake
- Stick 1, Trigger (held down): shoot
- Stick 1, Button 4-5: conveyor forward/backward

- Stick 2, Button 4-5: conveyor in/out AND intake rollers in/out
- Stick 2, Button 2: extend intake OR go to retry state (override) / Low Goal Routine 
- @deprecated Stick 2, Button 10-11: choose low goal/high goal
- Stick 2, Trigger: spin up shooter // Automagic 
- Stick 2, Button 3: High Goal Routine 
- Stick 2, Button 8: reset state machine (DANGER)
- Stick 3, Button 2: arm override
- Stick 3, Button 3: intake override
